## About
[LΛMPLIGHT氏](https://lamplight0.sakura.ne.jp/a/) が公開している「シャサフ式音楽理論」に基づき、簡易的な音楽を作成できるWebシーケンサー。

音同士の音高差を視覚的に示す「和音図」を拡張した、ピアノロールのようなUI上で編集・再生が可能。

現在のバージョン：テスト版（testflight-2025-07-30） バグを含む可能性があります。
