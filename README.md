# Nafchanaphata
Shasavistic Music Sequencer on Web

Available on Github Pages
[Nafchanaphata](https://rtt398.github.io/nafchanaphata/)

## About
[LΛMPLIGHT氏](https://lamplight0.sakura.ne.jp/a/) が公開している「シャサフ式音楽理論」に基づき、簡易的な音楽を作成できるWebアプリ
