工程

快捷键与帮助

播放

Space — 播放 / 暂停

Skip 按钮 — 跳回开头

底部循环开关 — 循环播放

音符操作

点击网格空白 — 创建根音符

点击音符 — 打开编辑弹窗

拖拽音符左端 — 改变起始位置

拖拽音符右端 — 改变长度

拖拽音符中部 — 改变音高

弹窗 delete 按钮 — 删除音符

弹窗 visibility 按钮 — 隐藏/显示音符（Ctrl+点击仍可选中）

Extend — 向当前音符添加泛音子音符

Trans — 将当前音符转写到另一维度

Prog — 累进移动根音音高

弹窗 Hz 栏可直接编辑频率

弹窗内可调音量、静音、粗细、透明度

扩展快捷键

弹窗底部显示各维度的快捷按键

点击快捷键标签可自定义键位（支持双键组合如 A+B）

未设置快捷键的维度显示"自定义"，点击即可设置

框选与批量操作

Shift + 拖拽空白 — 矩形框选

拖拽任一选中音符 — 整体移动组

点击空白 — 取消选区

Ctrl + C — 复制选中

Ctrl + V — 粘贴（原位过近自动偏移）

Ctrl + X — 剪切

Ctrl + A — 全选

Ctrl + D — 删除选中

自定义维度（Config）

Config → 自定义维度 → 扳手图标 — 打开管理弹窗

输入分子/分母（如 7/6），预览连线颜色和竖线位置

添加后在扩展/进行/转写菜单中出现对应按钮

支持任意有理数比率（n > d > 0）

自定义维度可通过勾选框独立启用/禁用

Config 面板

拍长（ms）/ 音高 / 分辨率 / 谱线维度 / 高维度

分割谱线 — 按分辨率细分拍内谱线

时值相同 — 点击创建音符时值自动匹配分辨率

卷帘式播放 — MIDI 风格滚屏

音符颜色 — 波长 / HSI增强 / 白色

粗细 / 连线粗细 / 连线透明度 — 全局或单音调节

视图缩放

Ctrl + 滚轮 — 等比缩放

Ctrl + Shift + 滚轮 — 纵向拉伸

Ctrl + Alt + 滚轮 — 横向拉伸

时值分辨率

Tab + 数字 — 切换分辨率（如 Tab+4 → 1/4，Tab+12 → 1/12）

支持两位数（Tab 后连续输入，800ms 超时自动提交）

按 0 忽略，再次按 Tab 立即提交

编辑辅助

Ctrl + Z — 撤销

H — 打开此帮助

左下角 Shift 开关 — 连带编辑模式

左下角垃圾桶 — 一键清屏

导出

MIDI（微分音） — 全微分音 MIDI 导出（含 pitch bend）

MIDI（12平均律） — 量化到 12-TET

MIDI（自定义律制） — 输入 EDO 值，导出为对应律制

WAV — 音频渲染导出

导入

MIDI（12平均律） — 含 pitch bend 还原

MIDI（微分音EDO） — 按指定 EDO 映射，自动检测轨道名

工程文件 — 加载 .naf 文件

工程管理

保存 / 加载 .naf 工程文件

URL 分享 — 序列化所有音符和设置

保存时可选路径和文件名
